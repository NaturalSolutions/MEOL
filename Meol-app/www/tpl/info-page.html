Rules of the game

  • Calculating the level of a collection :

    • Computing : averages or medians

      • Done : Number of taxons per collection
      • Potential :

        • Weighting of taxonomic groups
      • Test on new collections:

    • Human factor

      • in the body of the description of the collection..

  • Scoring

    • Score calcul : If the aswer is correct the player win X points weighted by following variables :
      • number of continents of selected taxon
      • IUCN Status weight of selected taxon. The weight is based on Fibonacci number, the assumption is : taxa with status are more known.
      • Total number of possible correct answers to the question
    • Deck example 
                            x points
                       + x1 / number of continents
                       + x2 / IUCN weight ( 2 = no status; 3 = LC ; 5 = NT; 8 = VU ; 13 = EN ; 21 = [CR; EW; EX])
                       + x3 / number of possible correct answers
                      = Nb of won points
    • Bonus : bbased on the number of correct aswers following (Fibonacci number).
      The player win bonus starting from 3 correct answers then each new response make him win points. 
                          [0;3[ = 0 * x
                    3 = 3 * x
                    4 = 5 * x
                    5 = 8 * x
                    6 = 13 * x
                    7 = 21 * x
                    8 = 34 * x
                    ...
    • Number of won points during a game : Points + Bonus In real time the progress bar goes to be present (bonus + point by deck) when the user exceeds the unlocking => message indicating that he has released a collection in which it will play once it has finished its part (end of part decided by the user)
    • Game example 
                          For each deck
                    Good answer : 1000 points
                     + if number of continent >1 then number of continent *-10
                     + if IUCN weight >2 then IUCN weight *x (not done)
                     + if number of possible correct answers >1 then number of possible correct answers *-50
                    Bonus (x = 100)
                    For example the player do :  3 correct aswers following, loose, then 5 correct aswers following
                    ( 3*100) + (3*100) + (5*100) + (8*100) => 1900 points for the game
