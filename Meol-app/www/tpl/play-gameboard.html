    Rules of the game

    Scoring

    If the aswer is correct the player win 1000 points weighted by following variables :

    • number of continents of selected taxon
    • IUCN Status (Conservation status) weight of selected taxon. The weight is based on Fibonacci number, the assumption is : taxa with status are more known.
    • Total number of possible correct answers to the question
    • Deck example 
      1000 points
- 10*number of continents
- 10*IUCN weight ( 2 = no status; 3 = LC ; 5 = NT; 8 = VU ; 13 = EN ; 21 = [CR; EW; EX])
- 50*number of possible correct answers
= Nb of won points

    Bonus

    based on the number of correct aswers following (Fibonacci number).
    The player win bonus starting from 3 correct answers then each new record series make him win points.

    [0;3[ = 0 * x
3 = 3 * 100
4 = 5 * 100
5 = 8 * 100
6 = 13 * 100
7 = 21 * 100
8 = 34 * 100
         ...

    Number of won points during a game

    Points + Bonus

    In real time the progress bar goes to be present (bonus + point by deck) when the user exceeds the unlocking => message indicating that he has released a collection in which it will play once it has finished its part (end of part decided by the user)
